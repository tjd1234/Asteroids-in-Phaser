## Asteroids in Phaser

This is a sample
[Asteroids](https://en.wikipedia.org/wiki/Asteroids_(video_game))-like game
using [Phaser](https://phaser.io/).

It's mainly to learn about game software design and the [Cursor AI
editor](https://www.cursor.com/).
